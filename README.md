# Visual Product Search Engine
VR Course Final Project

## Folder Structure
```
project/
├── app.py                        # Streamlit demo app (Deliverable 2)
├── eval.py                       # Batch evaluation script (Deliverable 3)
├── requirements.txt
├── README.md
├── checkpoints/
│   ├── clip_finetuned.pt         # ← REPLACE with your trained CLIP weights
│   ├── hnsw_index.bin            # ← REPLACE with your built HNSW index
│   └── index_metadata.pkl        # ← REPLACE with your gallery metadata
└── captions/
    └── captions.json             # ← REPLACE with your generated captions
```

## Files to Replace

| Dummy file | What to put there | How it's generated |
|---|---|---|
| `checkpoints/clip_finetuned.pt` | Fine-tuned CLIP state dict | Save with `torch.save(model.state_dict(), ...)` after training |
| `checkpoints/hnsw_index.bin` | HNSW vector index | Built during offline indexing pipeline |
| `checkpoints/index_metadata.pkl` | List of `{"image_name": ..., "item_id": ...}` dicts | Built alongside the HNSW index |
| `captions/captions.json` | Dict mapping `"img/img/..."` → caption string | Generated by BLIP-2 captioner in offline phase |

## Running Batch Evaluation
```bash
# Ablation A (vision-only, no rerank)
python eval.py --ablation A --top_k 15

# Ablation B (frozen CLIP + BLIP rerank)
python eval.py --ablation B --top_k 15 --use_rerank

# Ablation C (fine-tuned CLIP + BLIP rerank)  ← best model
python eval.py --ablation C --top_k 15 --use_rerank

# Quick test on 50 queries
python eval.py --ablation C --top_k 15 --use_rerank --limit 50
```
